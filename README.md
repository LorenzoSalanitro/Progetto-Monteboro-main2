# Progetto Monteboro

Sito HTML di base pronto per lo sviluppo.

## Struttura

- `index.html` — pagina principale.
- `styles/style.css` — foglio di stile.
- `scripts/main.js` — script JavaScript di base.
- `assets/` — directory per immagini e risorse statiche.

## Uso

Apri `index.html` nel browser o usa un server di sviluppo locale.

## Note

Puoi aggiungere altre pagine HTML e modificare gli stili nella cartella `styles`.
